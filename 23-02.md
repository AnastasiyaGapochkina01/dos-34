1) Если не выполняли ДЗ по git - выполнить его
2) Создать себе ВМ с Ubuntu любым удобным способом (VirtualBox, VmWare, Parallels, облачный сервер)
3) Настроить доступ к ВМ по ssh любым способом - login/pass или ssh-keys
4) На ВМ выполнить задачи
- установить утилиту tree
    <details>
  <summary>решение</summary>
  
  ```bash
  sudo apt install -y tree
  ```
  
    </details>

- проверить, что она установлена и где
    <details>
  <summary>решение</summary>
  
  ```bash
  whereis tree
  ```
  
    </details>
- вывести корневое дерево каталогов
    <details>
  <summary>решение</summary>
  
  ```bash
  tree -L 1 /
  ```
  
    </details>

- в домашней директории своего пользователя создать каталог `dvps-tasks`
    <details>
  <summary>решение</summary>
  
  ```bash
  mkdir ~/dvps-tasks
  ```
  
    </details>

- в каталоге `dvps-tasks` создать еще каталог `linux_01`
    <details>
  <summary>решение</summary>
  
  ```bash
  mkdir ~/dvps-tasks/linux_01
  # или
  cd ~/dvps-tasks
  mkdir linux_01
  ```
  
    </details>

- в каталоге `~/dvps-tasks/linux_01` создать файл `text-00` с содержимым
    ```text
        this is text file in Linux
        change me/move me/copy me/delete me
    ```
    <details>
  <summary>решение</summary>
  
  ```bash
  mkdir ~/dvps-tasks/linux_01
  # или
  cd ~/dvps-tasks
  mkdir linux_01
  ```
  
    </details>

- выяснить, в каком каталоге вы сейчас и вывести его содержимое
    <details>
  <summary>решение</summary>
  
  ```bash
  pwd
  ls
  ```
  
    </details>
- перейти в корень файловой системы, убедиться что вы действительно там и вернуться в домашний каталог
    <details>
  <summary>решение</summary>
  
  ```bash
  cd /
  pwd
  cd ~
  # или просто
  cd
  ```
  
    </details>
- перейти в `/var/log`, вывести его содержимое в виде списка без подробностей
    <details>
  <summary>решение</summary>
  
  ```bash
  cd /var/log
  ls -1
  ```
  
    </details>
- из `/var/log` перейти в `/tmp` по относительному пути
    <details>
  <summary>решение</summary>
  
  ```bash
  cd ../../tmp
  ```
  
    </details>
- вернуться в домашний каталог (решение уже есть выше)
- перейти в каталог `/usr/share`, используя относительные пути, затем из `/usr/share` перейти в `/etc/ssh` так же используя относительные пути; далее перейти в `/usr/bin` любым путем и оттуда вернуться в домашний каталог
